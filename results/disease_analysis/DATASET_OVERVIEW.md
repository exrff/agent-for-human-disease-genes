```

│   ├── system_scores.csv    # 系统级ssGSEA得分

│   └── [其他标准化数据文件]
└── analysis_results/    # 深入分析结果
    ├── [分析报告].md
    ├── [可视化图片].png

```

## 使用说明

### 干净数据 (clean_data/)
- 这些是标准化的、可直接用于分析的数据文件
- 所有文件都遵循统一的格式规范
- 适合用于进一步的统计分析或可视化

### 分析结果 (analysis_results/)
- 包含深入的生物学解释和临床意义分析
- 可视化图表和统计报告
- 特定疾病的专门分析结果
